# SLMFIX-Leveraging-Small-LLMs-for-Error-Fixing-with-Reinforcement-Learning
SLMFix — Uses PPO-based RL to train Qwen-2.5-Coder-0.5B to fix broken SQL and Bash commands generated by a larger proposer model. Broken candidates are detected via SQLGlot and bashlex validators, rewarded using AST and token-level F1 similarity, and evaluated on Spider and NL2Bash benchmarks.
